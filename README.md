# ✨ Overview

**DataLens** is a modern browser-based Exploratory Data Analysis (EDA) platform designed to simplify dataset analysis using AI-powered automation and interactive visual interfaces.

The platform helps users:

* 📊 Understand datasets instantly
* 🧹 Clean and preprocess data automatically
* ⚙️ Generate engineered features
* 📈 Create interactive visualizations
* 🧠 Receive AI-generated insights
* 📄 Export professional reports

Built entirely using frontend technologies with a lightweight single-file architecture.

# 🔥 Core Features

## 📥 Smart Dataset Analysis

DataLens automatically analyses uploaded datasets and detects:

* Column types
* Missing values
* Statistical summaries
* Schema information
* Dataset metadata
* Quality issues
